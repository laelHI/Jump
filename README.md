# Jump

A simple jumping game where you jump over blocks to increase your score. Hit a block, you die

# Features:

Simple Jump game with:
- a blue box representing you
- moving red blocks that you need to jump over
- cool features when you die
- jumps that rotate
- ... and you can play it all with one key!

# Technicalities 
- Javascript
